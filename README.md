## Key Features
- Authentication
- Group calls
- Share screen
- Record calls
- View call stats
- Instant meetings
- Schedule meetings
- Personal Meeting Room
- Join meetings via link
- View Upcoming & Previous meetings

## Tech Stack
- Next.js 14
- Typescript
- Tailwind CSS
- Shadcn UI
- Clerk
- Stream
- React Datepicker
